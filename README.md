## 🧭 Overview

EFRD is a Windows app for fiscal simulation and public planning. It models tax rules with a smooth curve instead of fixed tax bands. It also estimates a minimum lifetime wage, adjusts support for family load, and checks budget balance with simple math rules.

This project is built for users who want to explore a fiscal model without dealing with code. You can run it on a Windows PC, enter a few values, and review the results in a clear format.

## 📊 What EFRD does

EFRD uses a fiscal model that replaces tax brackets with one smooth curve. That helps the system avoid sharp jumps when income changes.

It can help you:

- Estimate a minimum lifetime wage
- Adjust support based on family load
- Review how inflation affects results
- Check if the model stays within a zero-deficit rule
- Compare different fiscal settings in one place

The app is useful for people who want a clear view of how a policy model changes under different conditions.

## 🧮 Main ideas in the model

EFRD uses a few core ideas:

- A continuous tax curve instead of tax bands
- Family support based on OECD-style load factors
- Balance checks that aim for no deficit
- Simulation tools that help test different cases
- Math rules that keep the model stable

You do not need to understand the math to use the app. The interface is meant to guide you through the process.
